# Plataforma-Streaming-
Playlist de mídias digitais, contendo diferentes tipos de conteúdo, como música, podcast e vídeo curto.

Em grupos de até 5 integrantes, vocês deverão desenvolver, em PHP, um pequeno aplicativo de linha de comando inspirado em plataformas de streaming utilizadas no seu dia a dia, como Spotify, YouTube, TikTok ou Netflix.

O sistema deverá simular uma playlist de mídias digitais, contendo diferentes tipos de conteúdo, como música, podcast e vídeo curto. Para isso, o grupo deverá aplicar obrigatoriamente os conceitos de herança, interfaces e polimorfismo.

A implementação deverá utilizar sempre:

declare(strict_types=1)
atributos privados
promotor de propriedades no construtor 
tipagem adequada
A  entrega deverá ser feita em um repositório no GitHub.



Requisitos obrigatórios
O projeto deve conter:

Uma classe abstrata Mídia, contendo atributos e comportamentos comuns.
Pelo menos 3 classes filhas (Musica, Podcast, VideoCurto)
Uma interface chamada Reproduzivel, co o método reproduzir(): string
Uso de polimorfismo, permitindo que diferentes tipos de mídia sejam reproduzidos por meio da mesma interface.
Uma classe Playlist, responsável por armazenar mídias e reproduzir todos os itens.
Entrega no GitHub.
